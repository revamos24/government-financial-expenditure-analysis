# Government Financial Expenditure Analysis Dashboard

## Project Overview

This project analyzes government financial expenditure using Microsoft Excel and Microsoft Power BI. The objective is to transform raw financial transaction data into interactive dashboards that support transparency, accountability, and data-driven decision-making.

## Tools Used

- Microsoft Excel
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

## Key Features

- Executive Dashboard
- Operational Dashboard
- KPI Cards
- Interactive Slicers
- Monthly Expenditure Trend
- Ministry Analysis
- Regional Analysis
- Funding Source Analysis
- Departmental Analysis
- Project Status Analysis

## Files Included

- Government Expenditure Dashboard (.pbix)
- Government Financial Expenditure Analysis Project (.pdf)
- Government Financial Expenditure Analysis Project (.docx)
- Cleaned Dataset (.xlsx)
- Raw Dataset (.xlsx)

## Key Insights

- Total government expenditure exceeded ₦6 billion.
- Internally Generated Revenue (IGR) was the largest funding source.
- The South East recorded the highest expenditure.
- Human Resources recorded the highest departmental expenditure.
- Interactive dashboards enable dynamic analysis using filters and slicers.
